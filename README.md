# Gatry Web Scraping

Simple web scraping on [Gatry Website](https://gatry.com/), saving the title and the price of items on a MongoDB collection

## Technologies

* 🧠 NodeJS
* 💻 Selenium WebDriver
* 🍃 Mongodb

## Usage

```bash
node index.js
```
